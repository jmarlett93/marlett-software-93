Prime Number Generator: Putting Web Workers To Work

In this simulation we will run a costly function to count the number of prime numbers within a large range. Performing this operation on the main thread will slow down the UI, and make it unresponsive. Instead we will use a webworker to take the calculation off the main thread keeping our site nice and responsive. Read more about web workers here
